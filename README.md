## GETTING STARTED

### Installation

#### Clone the project
```
https://github.com/Daniel151505/rick-and-morty-finder.git
```

#### Install node modules with npm
```
cd rick-and-morty-finder
npm install
```
#### Run Local
```
npm run serve
```
#### Run and compile build for production
```
npm run build
```
#### Run test - Cypress
```
npm run test:e2e
```
#### Run lints
```
npm run lint
```
